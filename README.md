# bun-api-hono

A lightweight Bun API server using Hono

## Features

- Built with [Hono](https://hono.dev/)
- TypeScript support
- Clean, minimalist API design

## Installation

To install dependencies:

```bash
bun install
```

## Usage

To run in development mode with hot reloading:

```bash
bun dev
```

To run in production mode:

```bash
bun start
```

Or run directly:

```bash
bun run index.ts
```
